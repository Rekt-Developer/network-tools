## 📊 Commands

| Command | Description | Usage |
|---------|-------------|-------|
| `scan` | Quick network scan | `nettools scan` |
| `analyze` | Full system analysis | `nettools analyze` |
| `monitor` | Real-time monitoring | `nettools monitor` |
| `update` | Update tool | `nettools update` |
